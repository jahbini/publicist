# Agents Roadmap

This directory is reserved for an experimental agent layer that remains
separate from the current runner behavior.

## Phase 1

Draft-only publicist. The agent can plan, prepare drafts, and write proposed
intents into Memo-backed artifacts without performing live operations.

## Phase 2

Contact ledger. Introduce a reviewed ledger for contacts, status, approvals,
and follow-up state.

## Phase 3

HTTP lookup through an allowlisted device. Network access remains indirect,
constrained, and routed through approved devices with Memo visibility.

## Phase 4

Reviewed send, log, and follow-up. Outbound actions stay behind explicit human
review before any public outreach or posting occurs.
